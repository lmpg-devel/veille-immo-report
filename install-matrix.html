Diagnostic installation Veille Immo

Version 1.7. Ces APK couvrent les 3 packages historiques et les 2 signatures possibles.

Package mobile: be.veilleimmo.mobile

A essayer en premier si l'app installee est recente.

mobile release mobile debug

Package watch: be.veilleimmo.watch

A essayer si l'app vient des versions 1.2 ou 1.3.

watch release watch debug

Package app: be.veilleimmo.app

A essayer si l'app vient du tout premier APK.

app debug app release
Ferme Veille Immo depuis le selecteur d'applications Android avant installation. Si les six boutons echouent avec "application non installee", Android refuse la mise a jour car l'application installee utilise une signature inconnue. Dans ce cas, la seule solution Android fiable est de desinstaller l'ancienne app puis installer l'APK courant.
